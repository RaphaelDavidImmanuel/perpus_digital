# Perpustakaan Digital

Aplikasi Perpustakaan Digital berbasis Laravel.

## Fitur
- Login dan Register
- Dasboard Admin
- Kelola Kategori Buku
- Kelola Buku (ada cover buku, judul, tahun, penulis, penerbit)
- Melihat Buku terpopuler, dan ada card statistiknya

- Dasboard User
- Buku terbaru yang ada
- Riwayat bacaan
- lihat katalog buku (fitur search buku, filter kategori)
- detail buku
- Baca buku online
- Download buku (PDF ONLY)

## Teknologi
- Laravel
- MySQL
- Bootstrap
- JavaScript

composer install
cp .env.example .env
php artisan key:generate
php artisan migrate

npm install -g npm@11.12.1
npm run dev
php artisan serve
